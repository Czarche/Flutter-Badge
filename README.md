# badge

A Flutter widget that help you to create badges

<p align="center">
  <img src="https://i.imgur.com/qMObRQc.png" alt="badges" style="margin:auto">
</p>




## Getting Started
### import 

```dart
import 'package:flutter/badge.dart';
```

### Example

```dart
new Badge.before(
              value: "Text",
              child: new Text("button"))
```
